Notes — Quick Guide
Notes is a personal notebook inside the Akasha knowledge system. Write in chunks — each chunk is an independent, versioned piece of text that can be reordered, edited, and linked to concepts in the graph.
Basic Workflow
  1. Open or create a note from the Library
  2. Add chunks to the document
  3. Click any chunk to edit it
  4. Reorder chunks by dragging in the left panel
  5. Use Undo / Redo to recover earlier versions
Chunks
A chunk is an independent block of text — a paragraph, a list, a quote, or a fragment. Each chunk is versioned separately. The left panel shows all chunks as a list you can reorder by dragging.

Philosophy
Notes is intentionally minimal. Its purpose is not to replace a word processor or publishing tool. It is a thinking space — a place to accumulate fragments, drafts, observations, and ideas as part of a living semantic graph. The chunk structure encourages short, revisable units rather than long monolithic documents.
